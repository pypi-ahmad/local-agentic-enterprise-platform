# Beginner Guide

## What This Platform Does

It automates office work using multiple AI agents:
- Read and summarize emails
- Plan calendar actions
- Generate reports
- Analyze KPIs
- Query databases safely
- Build multi-step business workflows

## How It Works

1. You submit a request in UI/API.
2. Supervisor agent chooses specialist agent(s).
3. Model router selects best local Ollama model.
4. Workflow executes with retries/approvals.
5. Results saved to database and shown in dashboard.

## First Learning Path

1. Log in from Dashboard.
2. Open AI Assistant and run a task.
3. Create a report and export it.
4. Upload a document and query Knowledge Base.
5. Generate SQL and inspect safety explanation.
